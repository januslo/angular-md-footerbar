Angular Footerbar

This Angular directive will place a bar at the bottom of your webpage. Perfect for a mobile web page! You can add icons if you like but not necessary.

Bower Install:

$ bower install footerbar

Add the footerbar.js to your index.html

<script src="js/directives/footerbar.js"></script>

Add directive to your main module:

Angular Material is a requirement so make sure you add this to your project.

(function () {
    angular.module('app', ['ngMaterial', 'ngAnimate', 'ngAria', 'footerBar']);
})();

Create Footer Items:

Create a list of footer items (or whatever you want to call it) in your controller. I use vm.footeritems but you can use $scope.footeritems of course.

vm.footeritems = [
            {
                name: 'Home',
                icon: '',
                size: {
                    size: 'md-48',
                    padding: 10
                },
                style: {
                    fill: 'white'
                }
            },
            {
                name: 'Website',
                icon: '',
                size: {
                    size: 'md-48',
                    padding: 10
                },
                style: {
                    fill: 'white'
                }
            }
];

The above will create a footer bar with only 2 menu items whose text is white and size is 48px labeled "Home" and "Website" with no icons.

Configure HTML

place the following somewhere in the body of your index.html:

<footer-bar items="vm.footeritems"></footer-bar>

Apply CSS:

Make sure you have material-design.css and bootstrap.css loaded. Put the following content in your CSS (or use the one provided in the repository):

.md-inline-list-icon-label{
    color:white;
}
#footerbar{
    width:100%;
    bottom: 0pt;
    padding-left:0px;
    margin-left:0px;
    position:fixed;
    left:0px;
    z-index: 100;
}
#footertable {
    margin-bottom:0px;
}

Options

barcolor specifies the color of the complete footer bar (default:blue):

<footer-bar items="vm.footeritems" barcolor="#6170B5"></footer-bar>

Icons

To add icons, use an iconprovider:

(function(){
angular.module('app')
    .config( function( $mdIconProvider ){
    $mdIconProvider.icon("magnify", 'icons/magnifying.svg', 128);
  });
})();

Now i can use the svg icon by name when setting up my footer items:

vm.footeritems = [
            {
                name: 'Home',
                icon: 'magnify',
                size: {
                    size: 'md-48',
                    padding: 10
                },
                style: {
                    fill: 'white'
                }
            },
            {
                name: 'Website',
                icon: '',
                size: {
                    size: 'md-48',
                    padding: 10
                },
                style: {
                    fill: 'white'
                }
            }
];

In the above, only the "Home" menu item will have a magnifying glass icon which has nothing to do with "Home" of course!
