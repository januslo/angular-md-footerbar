Angular Material Footerbar

This Angular directive will place a bar at the bottom of your webpage. Perfect for a mobile web page! You can add icons if you like but not necessary.

Demo Preview:

Add the footerbar.js to your index.html

<script src="js/directives/footerbar.js"></script>

Add directive to your main module:

Angular Material is a requirement so make sure you add this to your project.

(function () {
    angular.module('app', ['ngAnimate', 'ngAria', 'ngMaterial', 'footerBar']);
})();

Create Links In Your Footer:

Create a list of footer items in your controller. These will be shown as text buttons in your footer bar. I use vm.footeritems but you can use $scope.footeritems or similar.

vm.footeritems = [{
    name: '广州',
    icon: 'location_on',
    size: '32',
    padding:10,
    href:'#/index',
    isActive:true,
    iconStyle: {
        display:'block',
        width:'100%'
    }
},
    {
        name: '订单',
        icon: 'receipt',
        size: '32',
        padding:10,
        iconStyle: {
            display:'block',
            width:'100%'
        }},
    {
        name: '我',
        icon: 'timer_auto',
        size: '32',
        padding:10,
        href:'#/setting',
        iconStyle: {
            display:'block',
            width:'100%'
        }
    }];

The above will create a footer bar with only 2 menu items whose text is white and size is 48px labeled "Home" and "Website" with no icons.

Configure HTML

place the following somewhere in the body of your index.html:

<footer-bar items="footeritems" barcolor="#ffffff" activebarcolor="#ffffff" lineheight="28px" color="grey" activecolor="orange"></footer-bar>
